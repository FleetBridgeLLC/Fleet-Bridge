PROJECT TITLE
FleetBridge — Industrial Equipment Exchange Platform

OBJECTIVE
Build a clean, professional industrial equipment brokerage website for FleetBridge. The website should present FleetBridge as a serious marketplace platform connecting buyers and sellers of industrial equipment such as forklifts, trailers, warehouse equipment, and construction machinery.

TECH STACK
Use the following stack:

• HTML5
• CSS3
• Vanilla JavaScript
• Responsive design (mobile friendly)
• Single-page layout with modular sections
• Clean folder structure

Do NOT use heavy frameworks.

The website must run easily as a static site.

Body: Inter Regular

Brand Assets:

Logo file: fleetbridge-logo.png
Favicon: fleetbridge-favicon.png

The header must display the FleetBridge logo.

DESIGN STYLE

The visual style must look like an industrial platform used by logistics or equipment companies.

Design characteristics:

• wide layout
• strong typography
• large industrial imagery
• clean card layouts
• professional color usage
• strong section spacing

Avoid playful startup aesthetics.
